What is SBOLCanvas?

SBOLCanvas is a web application for creation and editing of genetic constructs using the SBOL data and visual standard. SBOLCanvas allows a user to create a genetic design from start to finish, with the option to incorporate existing SBOL data from a SynBioHub repository.

saSBOL Application Functionality

Previously, depending on the assembly standard (Biobrick, Berkeley, Silver, Freiburg) and Assembly Method (TypellS, 3A, Direct DNA synthesis, Gibson Assembly), scar sequences had to be manually input inbetween the parts such as the promoter, ribosome binding site (RBS), and coding sequence (CDS). However, more research regarding scar sequences has led to predictable behavior depending on which assembly method is used. Now, synthetic biologists using SBOL only need to design their target sequence, and once they export that sequence into an SBOL2 file (.xml), they can import their design into our app. Our tool will then suggest a recommended assembly standard, and once the designer chooses a standard, a sequence accurate SBOL2 file with scars and/or overhangs will be able to be downloaded.
